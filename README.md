# climp-website

Marketing/demo website for [climp](https://github.com/olivier-w/climp), a CLI music player.

## Stack

- [Astro](https://astro.build/) - static site generator
- [React](https://react.dev/) - interactive player component
- [Tailwind CSS v4](https://tailwindcss.com/) - styling
- [Bun](https://bun.sh/) - package manager

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview  # preview production build
```

## Deployment

Deployed on [Vercel](https://vercel.com/).
